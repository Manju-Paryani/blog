## 使不同的Atom window具有不同的 project

有时会遇到重新开启window后，添加project后，左侧的project区域为空，这个时候只要点击:

```highLight
View->Developer->Reload Window（Ctrl+Shift+F5）
```

即可使 project 目录在左侧project区域正确显示。

注：在新版本已无该问题。
